# LOLA LDEM Utilities for LANS-AFS-SIM Multipath Extension

This repository contains utility programs for reading LOLA LDEM data and generating a crater-rim elevation mask for the multipath and diffraction models used in the experimental `multipath` branch of [LANS-AFS-SIM](https://github.com/osqzss/LANS-AFS-SIM).

The main purpose of this directory is to provide:

- a lightweight interface to LOLA LDEM GeoTIFF data,
- examples for obtaining local elevation and surface normal vectors,
- a tool for generating the azimuth-dependent elevation mask around Shackleton Crater.

## Directory Contents

```text
multipath/ldem/
  LDEM_80S_80MPP_ADJ.TIF       LOLA LDEM GeoTIFF file
  download_ldem.sh             Script to download the LDEM GeoTIFF file
  ldem.c                       LDEM reader and coordinate conversion functions
  ldem.h                       Header file for the LDEM utility functions
  ldem_example.c               Example program for querying elevation and surface normal
  shackleton_edge_mask.c       Tool for generating the Shackleton crater-rim elevation mask
  shackleton_edge_mask.csv     Example output of the crater-rim elevation mask
  Makefile.example             Makefile for ldem_example
  Makefile.shackleton          Makefile for shackleton_edge_mask
  shackleton_skyplot.py        Python script for visualising the elevation mask as a skyplot
```

## Required Package

These utilities use `GDAL` to read the LOLA LDEM GeoTIFF file and extract georeferencing metadata embedded in the GeoTIFF. On Ubuntu, install the development package as follows:

```bash
sudo apt install libgdal-dev
```

## Downloading the LOLA LDEM File

The LOLA LDEM GeoTIFF file is not generated by the source code. It should be downloaded from [NASA's Planetary Geodesy Data Archive](https://pgda.gsfc.nasa.gov/products/90) before building or running the examples.

Run:

```bash
./download_ldem.sh
```

This script downloads the following file:

```text
LDEM_80S_80MPP_ADJ.TIF
```

This file is the LOLA LDEM product used by the example programs in this directory.

## LDEM Reader

The files `ldem.c` and `ldem.h` provide basic functions for accessing the LOLA LDEM GeoTIFF data.

They are used to:

- open and read the LDEM GeoTIFF file using GDAL, extracting georeferencing parameters (origin, pixel size) from the embedded metadata,
- convert between latitude/longitude and DEM x-y coordinates,
- obtain the local elevation,
- estimate the local surface normal vector from the DEM.

These functions are used by the example programs and can also be integrated into the LANS-AFS-SIM multipath models.

## Query Elevation and Surface Normal

The file `ldem_example.c` demonstrates how to use the LDEM reader.

It can query the elevation and local surface normal vector at an arbitrary position specified either by latitude/longitude or by DEM x-y coordinates.

### Build

```bash
make -f Makefile.example
```

### Usage

Query by latitude and longitude:

```bash
./ldem_example LDEM_80S_80MPP_ADJ.TIF latlon -89.67 129.78
```

Query by DEM x-y coordinates:

```bash
./ldem_example LDEM_80S_80MPP_ADJ.TIF xy 7690.244 -6402.723
```

The program outputs the local DEM height and the estimated unit normal vector at the specified position.

## Shackleton Crater-Rim Elevation Mask

The file `shackleton_edge_mask.c` generates an azimuth-dependent elevation mask caused by the crater rim as viewed from a receiver located near the center of Shackleton Crater.

The output is a CSV file containing the crater-rim elevation angle over a full azimuth range.

This elevation mask will be used by the diffraction model in LANS-AFS-SIM to determine the line-of-sight blockage and diffraction conditions around the crater rim.

### Build

```bash
make -f Makefile.shackleton
```

### Usage

```bash
./shackleton_edge_mask LDEM_80S_80MPP_ADJ.TIF shackleton_edge_mask.csv
```

The generated CSV file has the following example name:

```text
shackleton_edge_mask.csv
```

This file is used as an input to the diffraction model when computing the range bias and power loss caused by crater-rim diffraction.

## Skyplot Visualisation

The file `shackleton_skyplot.py` visualises the crater-rim elevation mask as a skyplot.

### Usage

```bash
python shackleton_skyplot.py [input.csv] [output.png]
```

If arguments are omitted, the script reads `shackleton_edge_mask.csv` and writes `shackleton_skyplot.png` in the current directory.

Example:

```bash
python shackleton_skyplot.py shackleton_edge_mask.csv shackleton_skyplot.png
```
